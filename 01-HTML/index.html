Etiquetas de Bloque VS Etiquetas de Línea

La etiqueta de bloque por excelencia es la <div>, una etiqueta de bloque es aquella que genera saltos de línea con sus etiquetas hermanas.

La etiqueta de línea por excelencia es la <span>, una etiqueta de línea sólo ocupa el espacio necesario que tiene su contenido. Hola soy otra span. <img> <a> <hr> <br> <strong> <enfasis> <sup> <sub> también es una etiqueta en linea.

Imágenes

En HTML podemos utilizar diferentes tipos de imagen dependiendo de su compresión. Sin pérdida de calidad: GIF y PNG, o con pérdida de calidad como: JPG

GIF: formato sin perdida de capacidad para comprimir hasta 256 colores, válido para tintas planas, logotipos, etc... Acepta animaciones.

PNG-8: Igual a GIF y su ratio de compresión es mejor que el de un GIF. No acepta animaciones.

JPG: formato perfecto para cuatricromías e imágenes fotográficas, pero tiene perdida de calidad.

PNG-24: formato perfecto para fotografías y transparencias a color real, tiene muy poca perdida de calidad

SVG: Formato de imagen vectorial, sin pérdida y la más recomendable para logotipos y pictogramas.

Formularios

El elemento HTML <form> permite la recogida de datos para su posterior gestión por lenguajes dinámicos, dispone de elementos específicos para cada tipo de dato que queramos gestonar. Html no tiene capacidad para enviar esos datos ni para recogerlos.

Se utiliza un formulario HTML para rcoger la entrada del usuario. La entrada del usuario es La mayoría de las veces se envía a un servidor para su procesamiento.

Algunos se utilizan para navegar o puede darsele otra función
